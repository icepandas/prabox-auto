
> Prabox自动领取

## 工具介绍

- Python
- 手机或模拟器
- 相关依赖

`iOS`，请参考下面的配置：
- 使用真机调试wda，参考iOS 真机如何安装 [WebDriverAgent · TesterHome][1]
- 安装[openatx/facebook-wda][2]
- Python 3

## 依赖安装

``` bash
    pip install -r requirements.txt
```

## IOS手机操作步骤

1. 运行安装好的 `WebDriverAgentRunner`
2. 运行脚本
	* 命令行运行`python3  prabox.py`

> 睡觉也能领糖果…
